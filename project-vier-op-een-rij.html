Vier op een rij

Terug naar projecten Bekijk GitHub

Wat is dit project?

Vier op een rij is mijn game-project. De speler kiest een kolom, waarna de steen op de juiste plek in het bord terechtkomt.

Bij dit project lag de focus op arrays, game-logica, beurten wisselen en controleren of er vier dezelfde stenen naast elkaar staan.

Project Details

Doel

Een speelbare game bouwen met duidelijke wincondities.

Technieken

HTML, Bootstrap en JavaScript.

Resultaat

Een interactieve game waarbij spelers om de beurt zetten doen.
