Project

Collectables Web-app

Een webapplicatie voor het beheren en bekijken van verzamelobjecten.

Terug naar projecten Bekijk GitHub

Wat is dit project?

De Collectables Web-app is gemaakt om verzamelobjecten overzichtelijk te tonen en te beheren. Denk aan items met een naam, categorie, beschrijving en afbeelding.

In dit project heb ik geoefend met dynamische data, overzichten maken en nadenken over hoe gebruikers snel informatie kunnen vinden.

Project Details

Doel

Een duidelijke web-app maken voor verzamelobjecten.

Technieken

HTML, CSS, JavaScript en PHP.

Resultaat

Een overzichtelijke applicatie waarin collectables centraal staan.
